# 🏆 Create Tournament Feature

## 📄 Objetivo

Permitir al usuario crear un nuevo torneo o continuar uno previamente guardado.

---

## 💭 Requisitos funcionales

### ✅ RF-01 - Mostrar torneos guardados

Cuando existan torneos almacenados, el sistema debe mostrar una lista de torneos disponibles.

---

### ✅ ~~RF-02 - Estado vacío~~

Cuando no existan torneos almacenados, el sistema debe mostrar un estado vacío invitando a crear un torneo.

---

### 🚧 RF-03 - Crear torneo

El usuario debe poder iniciar la creación de un nuevo torneo.

---

### 🚧 RF-04 - Límite de torneos

El sistema debe permitir un máximo de 3 torneos almacenados simultáneamente.

---

### 🚧 RF-05 - Continuar torneo

El usuario debe poder seleccionar un torneo existente para continuar trabajando en él.

---

### 🚧 RF-06 - Eliminar torneo

El usuario debe poder eliminar un torneo almacenado.

---

## ℹ️ Información mínima visible

Cada torneo debe mostrar:

* Nombre
* Estado
* Fecha de creación
* Fecha de última modificación

---

## Estados del torneo

* draft
* active
* completed
